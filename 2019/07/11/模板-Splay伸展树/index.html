模板 Splay伸展树

发表于 2019-07-11 | 阅读次数：
本文字数： | 阅读时长 ≈

概况

伸展树（Splay Tree），是二叉搜索树中的平衡二叉树，为了防止树（$ O(logn) $）退化成链（$ O(n) $），引入了旋转、伸展操作。经过实践，这些操作还会方便程序工作。

这里介绍伸展树中普通平衡树的操作，以LUOGU-P3369【模板】普通平衡树的题目内容为实现目标，码风源于这篇Splay博客和这篇Treap博客的有旋Treap。

预备

同一般二叉搜索树类似，它的节点基本需要以下变量：

  • val。储存它的关键字。
  • ch[2]。储存它的两个儿子。ch[0]为左儿子，ch[1]为右儿子。
  • fa。储存它的父亲。
  • same。与这个节点关键字相等的数的数量。
  • size。以这个节点为根的子树中储存数（不是节点）的数量。通常为这个节点的same + 左子树的size + 右子树的size。

除此之外，一颗完整的树还需要以下变量：

  • root。这棵树的根节点。
  • cnt。可用节点的最大下标。即最后添加的节点的下标。通常以++cnt的形式创建新节点。

函数

以下共有11个函数。

void clear(int) - 清除数据

删除节点后调用。

int kind(int) - 左儿子还是右儿子

左儿子返回0，右儿子返回1。

void pushup(int) - 更新size数组

根据上文对size的描述更新size。注意此时左右儿子的size必须正确。

void rotate(int) - 旋转

分为左旋和右旋，k点作为右儿子进行左旋，反之。通过kind(k)即可轻松把左旋与右旋轻松合并为一个函数。注意处理好各个点之间的联系，尤其要注意祖父是否存在。

void splay(int) - 伸展到根

每次旋转k点都会上升，便可以把k伸展到根。能改善伸展树的结构。是伸展树的中心思想。要注意的是，k, fa[k], fa[fa[k]]的关系（是左儿子还是右儿子）不同，旋转的方式（左旋还是右旋，旋转几次）也不同。这里整合成一个for循环。

void insert(int) - 插入新值

非递归形式。根据二叉搜索树的性质插值。注意，根为0（树为空树）时，要特殊判定；不管是加入一个已有节点还是新创建节点，都要splay一次。

int find(int) - 查询排名

非递归形式，查找升序排列后它的排名（有多个数返回排名最小的）。和insert类似，小找左，大找右，相等特判。要注意查询右子树时答案要加上小于左子树和根的数的数量；找到相同的点后，在统计答案的同时也要splay。

int ith(int, int) - 排名为i的数

递归形式，查找第i小的数。三种情况如上一函数，同样要注意查询右子树时要更新i的值。不splay。

int pre(int) - v值的前驱

递归形式，返回点的关键字。查询比v小的数中最大的数（升序排列中排名为find(v) - 1的数）。思路希望读者自己推敲。

int suc(int) - v值的后继

同上。不解释。

int root_pre() - 根的前驱的下标

先找到根的左子树（所有比根小的数），再往右子树走到底（最大的数）。

void delet(int) - 删除v值

非递归形式。一开始要把它splay到根（splay的参数是数组下标，所以用find代替）。如果节点有多个数，只删除一个；否则删除节点。删除节点时，如果没有左右儿子，直接清理（为根时根赋0）；有一个儿子，儿子直接顶替它；两个儿子都有，用它的前驱顶替，可以通过splay把它的前驱移动到根处，原根就会位于叶子处，直接删去即可。
